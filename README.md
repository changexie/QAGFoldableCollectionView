# QAGFoldableCollectionView
### 使用CollectionView实现
#### 描述：
* 点击分组，分组进行折叠或者展开

* 默认第一组为当前正在编辑的分组，不能展开或者折叠，不能删除组中原有的成员

* 点击其他分组中的成员，会移动到第一组，这些成员在第一组中可以点击，点击过后回到原来的分组中
